    Overview

    API Endpoint

    https://tripledouble.herokuapp.com/api/v1/

    Welcome to tripledouble!

    You can use the free API to fetch season stats, player info, team rosters, injury reports, and scheduled games. No emails, sign-ups, or authentication required.

    Source code can be found here. Feel free to open any PRs should you see any missing features that you want to add.

    Considerations

    • Season stats and injury reports are updated daily at 2:00am PST
    • Scheduled games are updated on the first day of every month at 2:00am PST
    • Dataset only includes active players starting from the 2019-2020 season
    • API is rate limited to 3600 queries per hour
